PCのための無料でシンプルなプレイヤーアプリ

WindowsまたはmacOS搭載のPCで、お手持ちのmp3ファイルが再生できます。
基本的な操作は再生と停止、シークバーの操作、そしてトラックの移動のみ。
手軽さを重視したプレイヤーアプリです。

プレイリスト機能搭載

プレイリストを作成すれば、いつでも好きな順番で曲を再生できます。
必要な操作はファイルの追加・削除と保存のみ。

直感的なデザイン

リアルな描画を意識したニューモーフィズムデザインを採用。
また、スペクトラムアナライザを搭載しており、再生中にスペクトラムが表示されます。

動作環境

OS 推奨環境 CPU RAM
Windows Windows 10 以降(32bit/64bit) IntelまたはAMD 2.0GHz以上 8GB以上
macOS macOS 10.15 以降 Intel 2.0GHz以上 またはApple Silicon 8GB以上

仕様

対応音声ファイル形式 mp3, wav, ogg, m4a
対応画像ファイル形式（サムネイル画像） png, jpeg, bmp, gif, tga, ppm, pgm, pbm, pnm, webp
開発言語 C++
使用ライブラリ OpenSiv3D v0.6.3
